Alarm Clock for Synology

Alarm Clock – это программа, которая превратит ваше устройство DiskStation в интеллектуальный музыкальный будильник. В назначенное время вы будете разбужены тихими звуками вашей любимой мелодии.

Вы можете выбрать для воспроизведения любой плейлист, из настроенных в AudioStation, или создать специальный для этих целей.

Музыка может воспроизводиться через присоединенные к DiskStation USB - звуковую карту или динамики, а также на любом доступном из AudioStation сетевом проигрывателе по протоколу DLNA или AirPlay.

Вы можете настроить различные мелодии в разные дни недели, а также устанавливать несколько будильников на разные устройства. Например, рано утром в будни для вас, через присоединенные динамики, и чуть попозже – на сетевой проигрыватель в детской, для ребенка. А в выходные – установите будильник на более позднее время.

Для каждого будильника задается длительность воспроизведения.

Вы можете настроить плавное увеличение громкости воспроизведения, для того, чтобы ваше пробуждение было более приятным.

Основные настройки

Для запуска в назначенное время, AlarmClock использует встроенные средства планировщика задач DiskStation. Поэтому необходимо указать параметры учетной записи, с правами для добавления задач во встроенный планировщик. Например, это может быть учетная запись администратора системы.

Нажмите на кнопку «Настройки» на панели инструментов.

Откроется окно для ввода имени и пароля учетной записи.

Настройка будильников

Главное окно программы состоит из панели инструментов (1), списка задач (2) и панели настроек выбранной задачи (3).

  • Нажмите кнопку «Создать» для добавления новой задачи (будильника). В списке задач появится новая задача.
  • Выберите новую задачу в списке.
  • На панели настроек задачи вы можете настроить параметры выбранной задачи:
    • Выберите дни недели, когда будет срабатывать будильник. Задайте время.
    • Выберите список воспроизведения. Отображается перечень списков воспроизведения, настроенных в AudioStation. Вы можете создать (в AudioStation) специальный список воспроизведения для вашего будильника.
    • Выберите устройство для воспроизведения из списка устройств, доступных в AudioStation. Это может быть USB звуковая плата или USB динамики, либо сетевой проигрыватель с поддержкой протоколов DLNA или AirPlay.
    • Задайте длительность звучания (в минутах). Через указанное время воспроизведение остановится (и вы можете продолжить спать :) )
    • Настройте плавное увеличение громкости. Укажите начальный и конечный уровень громкости, и время, в течении которого громкость будет повышаться. Если заданное время меньше общей длительности звучания, то воспроизведение будет продолжаться с конечной громкостью все оставшееся время.
    • Все сделанные настройки задачи отобразятся в списке задач. Но еще не будут сохранены в настройках DiskStation.
  • Несохранные изменения выделяются красным индикатором в списке задач.
  • Для сохранения всех сделанных настроек, нажмите кнопку «Сохранить» на панели инструментов. Все сделанные настройки будут сохранены на диске DiskStation, и в основной планировщик задач DiskStation будут добавлены задачи для настроенных вами будильников.

  • Кнопка «Удалить» удаляет выбранную задачу из списка.

  • Кнопка «Сброс» служит для возврата списка задач к последнему сохраненному состоянию, если вы по неосторожности удалили нужный будильник или выбрали неверные настройки.

  • Кнопка «Запустить» запускает выбранный будильник прямо сейчас, чтобы вы могли проверить его работу, не дожидаясь настроенного времени. Если во время запуска произойдет ошибка, будет показано окно с информацией об ошибке.

Особенности

При воспроизведении на сетевом проигрывателе по протоколу DLNA, большинство проигрывателей автоматически включаются (переходят из режима сна) и начинают воспроизведение. Однако некоторые проигрыватели не включаются автоматически:

  • Yamaha AV Receiver

AlarmClock распознает проигрыватели из этого списка, и посылает им специальную команду для включения по сети.

Если у вас сетевой проигрыватель, который не включается автоматически при начале воспроизведения, напишите авторам программы, и он будет добавлен в список устройств, требующих включения по сети.
